# datanext

[DataONE](https://search.dataone.org) client using [next.js](https://github.com/zeit/next.js).

## Features

- [x] Package search
  - [x] Basic search filters
- [x] Package / metadata display
- [ ] Authentication
- [ ] Profiles
- [x] Serverless deployment
- [x] Standard node server deployment

## Running

```bash
npm install
npm run dev
```
